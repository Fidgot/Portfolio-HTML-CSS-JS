Etapes images voiture sans fond

1. Ouvrir paint 3D
2. Nouveau fichier
3. Bibliothèque 3D
4. Choisir image
5. Rotatation horizontale 90° vers la doirte
6. Redimensionner largeur 100%
7. Mettre l'image sur la ligne du bas
8.  Menu -> Enregistrer sous -> Image -> PNG -> Enregistre -> Dossier -> "voiture-{i}"
9. Fermer Paint 3D -> Ne pas Enregistrer
10. Aller sur https://www.remove.bg/fr
11. Télécharger l'image -> Télécharger -> Renommer "voiture-{i}-bg" -> Enregistrer
12. Créer le lien a href en HTML
